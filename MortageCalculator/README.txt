Description: 

Principal field denotes the amount borrowed. Interest field denotes the rate of interest.

Principal & interest values will always be positive and will not except -ve values.

No result will be displayed when term loan is not selected.

Taxes and Insurance: CheckBox that allows the user to select if taxes and insurance to be included in the monthly payment.

Calculate button will calculate the user's monthly payments based on values and using the formula.

TextLabel of result will display the monthly payment.

No additional settings are needed as per my testing.
